# Dataset

Due to GitHub's file size limitations, the full dataset (2GB) is hosted externally.

## Download the Dataset

**Full Dataset (2GB):**

Download from zenodo.org at this DOI: 10.5281/zenodo.17980746

Dataset Structure

After downloading,  put all the folders into this `data/`:
```
data/
├──├commonWaxbill/
│  ├ village_weaver/
│  ├ cisticola/
│  └── ...

```

## Dataset Description

- **Size:** 2GB
- **Format:** WAV audio files, 16kHz sample rate
- **Species:** 10 classes of bird species and 1 farm environmental sounds
- **Recording locations:** Bugesera, Busogo, Nyagatare (Rwanda)
- **Total samples:** 9970 samples
